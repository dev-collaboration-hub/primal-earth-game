# 🦣 Primal Earth

An open-source prehistoric survival action game featuring tribal warfare, wildlife, crafting, exploration, and immersive open-world gameplay.

---

# 📖 Overview

Primal Earth is a community-driven flagship open-world survival game set in a prehistoric world where players must survive against dangerous wildlife, harsh environments, rival tribes, and natural disasters.

Explore vast ancient landscapes, hunt massive creatures, craft primitive tools, build settlements, and lead your tribe through an evolving world.

---
